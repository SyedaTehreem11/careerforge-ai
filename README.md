# CareerForge AI

CareerForge AI is a polished, dark-mode SaaS-style experience built with React and Vite. It showcases a premium landing page alongside multi-step career tools for resume analysis, interview preparation, and job tracking.

## Highlights

- Premium landing experience with animated hero, feature sections, and testimonials
- Dedicated product views for dashboard, resume review, interview prep, and job tracking
- Responsive layout designed for a modern AI career platform feel
- State-driven interactions for simulated resume analysis and job pipeline management

## Tech Stack

- React
- Vite
- React Router DOM
- Framer Motion
- React Icons

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Project Structure

- src/App.jsx — Main app shell and route configuration
- src/pages — Dashboard, resume analyzer, interview prep, and job tracker views
- src/sections — Landing page sections such as hero, features, testimonials, and footer
- src/components — Shared UI elements and navigation
